Mon application DogCare est très fille pour les soins des chiens. 
Le but de l'application est de pouvoir noter la routine de pitou.
Comme ma clientèle c'est les girly pop 🎀, c'est pourquoi j'ai choisi un thème rose. 
Je suis allée chercher l'inspiration de palette de couleur sur Pinterest dans le thème coquette.


En ouvrant l'application, on y retrouve (les fonctionnalité ci-dessous):

Le titre, l'option Mode sombre. En touchant, il bascule sur une palette de couleur prune/rose foncé.
Il y a les bouton Ajouter promenade et réinitialiser qui active le compteur Promenades cette semaine.
Ensuite, il y a les autres Card qui affichent un message, mais il n'y a aucune action lorsqu'on les touche.

(Voici les détails sous-forme de points)

Fonctionnalités:
- Compteur de promenades hebdomadaires
- Bouton pour ajouter une promenade
- Bouton pour réinitialiser le compteur
- Switch pour activer/désactiver le mode sombre
- Liste de tâches affichée avec LazyColumn
- Utilisation de la programmation orientée objet (data class DogTask)


Pour lancer l’application:
1. Cloner le dépôt
2. Ouvrir le projet dans Android Studio
3. Appuyer sur Run
4. Utiliser le Switch pour changer le thème
5. Utiliser les boutons pour modifier le compteur
